# Платформа для создания тестов с ИИ

Это full-stack веб-приложение, представляющее собой платформу для создания и прохождения тестов. Ключевой особенностью является модуль на базе ИИ, который позволяет генерировать уникальные квизы по предоставленному пользователем тексту.

## Основные возможности

*   **Регистрация и авторизация:** Безопасная система входа с использованием JWT и рефреш-токенов.
*   **Генератор квизов на базе ИИ:** Пользователь может вставить любой учебный текст и получить готовый тест с вопросами и вариантами ответов (используется API OpenRouter).
*   **Интерактивное прохождение тестов:** Мгновенная обратная связь по каждому ответу.
*   **Таблица лидеров:** Результаты авторизованных пользователей сохраняются в общем рейтинге.
*   **Работа над ошибками:** После завершения теста доступна страница с полным разбором всех ответов.

## Технологический стек

*   **Бэкенд:** Python, Flask, SQLAlchemy, Flask-JWT-Extended
*   **Фронтенд:** React, TypeScript, Vite, Mantine UI, Axios
*   **База данных:** SQLite
*   **AI:** OpenRouter.ai API (модель Mistral 7B Instruct)

## Установка и запуск

### Бэкенд

1.  Перейдите в папку `backend`: `cd backend`
2.  Создайте и активируйте виртуальное окружение: `python -m venv venv` и `source venv/bin/activate` (или `venv\Scripts\activate` на Windows).
3.  Установите зависимости: `pip install -r requirements.txt`
4.  Создайте файл `.env` и добавьте в него ваши ключи:
    ```
    OPENROUTER_API_KEY=ВАШ_КЛЮЧ
    JWT_SECRET_KEY=ВАШ_СЕКРЕТНЫЙ_КЛЮЧ
    ```
5.  Запустите сервер: `python app.py`

### Фронтенд

1.  Перейдите в папку `frontend-ts`: `cd frontend-ts`
2.  Установите зависимости: `npm install`
3.  Запустите dev-сервер: `npm run dev`

Приложение будет доступно по адресу `http://localhost:5173`.
